\begin{theorem}

There is a fully faithful functor from the category of small multi-categories to the category of database multi-categories.

\end{theorem}

\begin{proof}

Let $\mcM$ be a small multi-category.  For any object in $M\in\Ob(\mcM)$, let $H(M)$ denote the set $$H_M:=\coprod_{\stackrel{n\geq 1}{X_1,\ldots,X_n\in\Ob(\mcM)}}\Hom_\mcM(X_1,\ldots,X_n;M).$$  Let $\DT_\mcM=\Ob(\mcM)$, let $U_\mcM=(\coprod_{D\in\DT_\mcM}H_D)$, and let $\pi_\mcM\taking U_\mcM\to\DT_\mcM$ denote the obvious map.

For each morphism $f\taking (X_1,\ldots,X_n)\to M$ in $\mcM$, let $S_f$ denote the operational table whose schema is $\sigma_f:=(X_1,\ldots,X_n;M)$ and whose row set is $$R_f:=\{(f_1,\ldots,f_n,g)\in H_{X_1}\cross\cdots\cross H_{X_n}\cross H_M\; |\; g=f\circ(f_1,\ldots,f_n)\}.$$  There is an obvious map $\delta_f\taking R\to\Gamma(\sigma_f)=H_{X_1}\cross\cdots\cross H_{X_n}\cross H_M$.  That is, we set $S_f=(\sigma_f,\delta_f)$.  Let $S_\mcM=\{S_f\;|\; f\in\mcM\}.$

In the following paragraph we will show that the pair $(\pi_\mcM,S_\mcM)$ is a database multi-category.  That is, it contains the identities, and is closed under operational table composition.  That it contains the identities is obvious: for any $M\in\Ob(\mcM)$ we find $S_{\id_M}$ in $S_\mcM$.

It is obvious that $(f_1,\ldots,f_n;f)$ is a composable sequence of arrows in $\mcM$. It is easy to see that this join is precisely $S_{f\circ(f_1,\ldots,f_n)}$.
