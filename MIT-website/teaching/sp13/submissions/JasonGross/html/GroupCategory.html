Problem

(The category Grp of groups). Suppose we set out to define a category Grp, having groups as objects and group homomorphisms as morphisms, see Definition 3.2.1.7. Show (to the level of detail of Example 4.1.1.4) that the rest of the conditions for Grp to be a category are satisfied.

  Definition CategoryOfGroups : @Category { T : Type & Group T }.
    refine {|
        
        Morphism := (fun s d : { T : Type & Group T } ⇒
                       GroupHomomorphism (projT2 s) (projT2 d));
        Identity := (fun x ⇒ identity_group_homomorphism _);
        Compose := (fun _ _ _ m1 m2 ⇒ compose_group_homomorphisms m1 m2)
      |};
    abstract (
        repeat intros [? ?]; intros;
        apply MonoidHomomorphism_Eq;
        reflexivity
      ).
  Defined.
End CategoryOfGroups.
