Problem

Let X = {a, b, c} and let M = ( ℕ, 1, × ) be the multiplicative monoid of natural numbers (see Example 3.1.3.2). Let f : X → ℕ be the function given by f(a) = 7, f(b) = 2, f(c) = 2, and let β_{X,M}: Hom_{Set}(X, R(M)) → Hom_{Mon}(L(X), M) be as in the proof of Proposition 5.1.1.2. What is β_{X,M}(f)([b, b, a, c])?
  Inductive X := a | b | c.
  Let M : Monoid ℕ := NaturalNumbersMonoid.
  Let f : X → ℕ
    := fun x ⇒ match x with
                  | a ⇒ 7
                  | b ⇒ 2
                  | c ⇒ 2
                end.

  Let β (X : Type) {T} (M : Monoid T)
    := InverseNaturalIsomorphism (AMateOf MonoidFreeUnderlyingAdjunction) (X, existT _ _ M).
  Arguments β X {T} M _.
  Eval compute in β X M f (b :: b :: a :: c :: nil).
End Exercise_5_1_1_3.
