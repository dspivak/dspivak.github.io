Problem

Let C = (G, ≃) be a schema. A leaf table is an object c : Ob C with no outgoing arrows. Write this condition mathematically in three different languages: that of graphs (using symbols V, A, src, tgt), that of categories (using Hom_C, etc.), and that of tables (in terms of columns, tables, rows, etc.).
In the language of categories, is there a difference between a terminal object and a leaf table?

  Section GraphLeafTable.
    Variable G : Graph'.
    Let V := Vertex' G.
    Let A := Arrow' G.
    Let src := Graph'Source G.
    Let tgt := Graph'Target G.

    Definition GraphLeafTable (v : V) := ∀ a : A, src a ≠ v.
  End GraphLeafTable.

  Section CategoryLeafTable.
    Let Ob {objC} (C : @Category objC) := Object C.
    Let Hom {objC} (C : @Category objC) := Morphism C.
    Context `(C : @Category objC).

    Definition CategoryLeafTable (c : Ob C) :=
      ∀ d : Ob C, Hom C c d ≅ (c = d).
  End CategoryLeafTable.
End Exercise_5_1_4_4.
