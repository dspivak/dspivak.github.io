Problem

Let T = (ℙ, η, µ) be the power set monad. The category Kls(T) is closed under binary coproducts, i.e. every pair of objects A B : Ob(Kls(T)) have a coproduct in Kls(T). What is the coproduct of A = {1,2,3} and B = {a, b}?

  Local Open Scope type_scope.

  Let T := PowerSetMonad.

  Let η := MonadUnit T.
  Let µ := MonadMultiplication T.

  Global Instance PowerSetMonadCoproducts X Y : is_coproduct (KleisliCategory T) X Y (X + Y).
  apply KleisliCategory_has_is_coproducts.
  exact (sumT_is_coproduct _ _).
  Defined.
End Exercise_5_3_3_7.
