Problem

Let S be a set and consider the indiscrete category K_S : Ob Cat on S objects (see Example 4.3.4.3).
(a) Does K_S have an initial object?
(b) Does it have a terminal object?

  Let K S : @Category S := IndiscreteCategory S.

  Variable S : Type.

  Example Exercise_4_5_3_13_a (s : S) : @IsInitialObject _ (K S) s.
  Proof.
    intro.
    ∃ tt.
    intros [].
    reflexivity.
  Defined.

  Example Exercise_4_5_3_13_b (s : S) : @IsTerminalObject _ (K S) s.
  Proof.
    intro.
    ∃ tt.
    intros [].
    reflexivity.
  Defined.
End Exercise_4_5_3_13.
