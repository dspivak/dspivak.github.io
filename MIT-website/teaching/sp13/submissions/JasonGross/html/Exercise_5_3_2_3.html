Problem

Let ℙ : Set → Set be the powerset functor, so that given a function f : X → Y the function ℙ(f) : ℙ(X) → ℙ(Y) is given by taking images.
(a) Make sense of the following statement: “with η defined by singleton subsets and with µ defined by union, T := (ℙ, η, µ) is a monad”.
(b) With X = {a, b}, write down the function η_X as a 2-row, 2-column table, and write down the function µ_X as a 16-row, 2-column table (you can stop after 5 rows if you fully get it).
(c) Check that you believe the monad laws from Definition 5.3.2.1.
